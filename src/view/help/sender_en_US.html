File Sender

A tool for automatically send lots of
files(attachments) throw your email
acount. This tool was made width the
purpose of share information in countries
with limited internet, not as a spaming tool.

Configuration

To configurate the File Sender go to
the Settings window and select the
Sender tab, you most configurate your
SMTP host there.
To get information about your mail host
you must to contact your mail provider or
write a mail to a wrong mail address and
look at the error message delivered by the
mail host, you may found something useful.
The SMTP port usually is 465, 25 or 587.
You can check if your configuration is
correct clicking the "test" button.

Usage

To use the File Sender click the "Send"
button, then provide one or more email
of a friend(s) to send the files, then
select the folder where are the files
you want to send, you can write the
folder path manually and then press
the "add" button or you can press the
"add" button (with the folder field in
blank) and select the folder with a
folder chooser.
You can use the clear button to clear
the file list.
When you are ready click the start button
to start sending the files, you can pause
the copy at any time.
In the notification area (the black box)
you can see notifications abut the state of
the copy.
The progress bar and the label to its right,
show the progress of the copy and useful
information abut the number of files.
You can delete selected files from the
copy list by pressing the "Delete" or "Del"
button on your keyboard, for security
reasons you must to pause the copy
before be able to delete any file
from the list.

If you have any problem using the Sender
please write me or you can send an Errors
Report using the button in the "Settings"
section (you can see my email address in
the "About" section)
